# Eight O(n²) Invariants Distinguishing All Groups of Order ≤ 63

## Overview

A set of **8 truly O(n²) pair-counting invariants** achieves **100% distinguishability** among all 319 groups of order ≤ 63. Every group has a unique 8-invariant signature.

The first collision occurs at order 64 (= 2⁶). See `more_groups_results.md` for the full analysis through order 200.

## Invariant Definitions

For each group, we compute 8 invariants by counting pairs (a,b) that satisfy:

1. **inv1**: ab = ba (commutativity)
2. **inv3**: a² = b² (same square)
3. **inv5**: a³ = b³ (same cube)
4. **inv6**: a⁴ = b⁴ (same fourth power)
5. **inv13**: (ab)² = e (product is an involution or identity)
6. **invA**: ab = ba AND a² = e (commuting pairs where a is an involution or identity)
7. **invB**: (ab)⁵ = e (product has order dividing 5)
8. **invC**: a⁷ = b⁷ (same seventh power)

## Distinguishability Results

- **319/319 (100%)** of all groups of order ≤ 63 have unique 8-invariant signatures
- First collision at order 64: SmallGroup(64,13) and SmallGroup(64,14)

## Legend

- **Order**: Group order (number of elements)
- **Index**: SmallGroup index
- **Group ID**: Standard SmallGroup notation
- **Abelian**: ✓ if abelian, ✗ if non-abelian
- **inv1**: Count of pairs (a,b) where ab = ba
- **inv3**: Count of pairs (a,b) where a² = b²
- **inv5**: Count of pairs (a,b) where a³ = b³
- **inv6**: Count of pairs (a,b) where a⁴ = b⁴
- **inv13**: Count of pairs (a,b) where (ab)² = e
- **invA**: Count of pairs (a,b) where ab = ba and a² = e
- **invB**: Count of pairs (a,b) where (ab)⁵ = e
- **invC**: Count of pairs (a,b) where a⁷ = b⁷

## Algorithmic Complexity

All 8 invariants are **truly O(n²)** using explicit double loops with O(1) work per pair:

```gap
for a in elements do
    a2 := a^2;
    for b in elements do
        ab := a*b;
        if ab = b*a then inv1 := inv1 + 1; fi;
        if a2 = b^2 then inv3 := inv3 + 1; fi;
        if a^3 = b^3 then inv5 := inv5 + 1; fi;
        if a2^2 = b^4 then inv6 := inv6 + 1; fi;
        if ab^2 = Identity(G) then inv13 := inv13 + 1; fi;
        if ab = b*a and a2 = Identity(G) then invA := invA + 1; fi;
        if ab^5 = Identity(G) then invB := invB + 1; fi;
        if a^7 = b^7 then invC := invC + 1; fi;
    od;
od;
```

No expensive GAP builtins (Subgroups, ConjugacyClasses, AutomorphismGroup, Order) are used. Each pair requires a bounded number of group multiplications and comparisons.
