# llm-research-sandbox

## Folders

- **gap-server/** — GAP HTTP server and deployment/installation notes for running GAP as a web service.
- **group-invariants/** — Scripts and results for computing and analyzing group-theoretic invariants using GAP and Python.
- **lean4-dockerfiles/** — Dockerfiles for building Lean 4 environments across multiple versions.
- **lean4web/** — Web frontend and server for running Lean 4 in the browser (fork of lean4web).
