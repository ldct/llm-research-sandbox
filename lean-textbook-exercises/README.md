# Lean 4 Textbook Exercises

Extracted exercise files from various Lean 4 textbooks and courses.
Each subfolder contains standalone `.lean` files with `sorry` placeholders to be filled in.

## Textbooks

| Folder | Source | Mathlib? | Exercises |
|--------|--------|----------|-----------|
| `tpil4/` | Theorem Proving in Lean 4 | No | 64 sorries across 12 files |

## Purpose

These exercises are used for testing LLM proof generation against a Lean/Mathlib REPL.
